What is Bootstrap?

Bootstrap is an essential tool for web developers. It is the most popular HTML, CSS, and JS framework for developing responsive projects.

Bootstrap’s Responsive Grid

Bootstrap comes with its own system of grid. It allows up to 12 columns across the page. Bootstrap's grid system is responsive, and the columns will re-arrange automatically depending on the screen size. Here how it works:

Images in Bootstrap are made responsive. It can change the size of images by using a class of img-circle and img-rounded. Bootstrap automatically does it for you.
